## Import Order

1. Jalankan `formula_none.sql` untuk membuat database, tabel, constraint, dan view.

## Regenerate Seed

1. Jalankan `python -m venv .venv-faker`.
2. Jalankan `.\.venv-faker\bin\python.exe -m pip install -r requirements.txt`.
3. Jalankan `.\.venv-faker\bin\python.exe generate_formula_none_seed.py`.
